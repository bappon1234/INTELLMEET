## 📌 Overview

IntellMeet is a **production-grade full-stack MERN application** designed to transform traditional meetings into **productive, actionable experiences** using AI.

It enables teams to:
- Conduct real-time video meetings
- Extract actionable tasks automatically
- Collaborate in real-time

---

## 🎯 Key Features

| Feature | Description |
|--------|------------|
| 🔐 Authentication | JWT-based login/signup system |
| 🎥 Video Meetings | Real-time meeting rooms |
| 📝 Action Items | Extract tasks from meetings |
| 💬 Real-time Chat | Live communication |
| 👥 Team Management | Invite & manage team members |
| 📊 Dashboard | Meeting history & insights |

## 🛠️ Tech Stack

### Frontend
- React + TypeScript + Vite
- Tailwind CSS + shadcn/ui

### Backend
- Node.js + Express
- MongoDB + Mongoose

### Real-time
- Socket.io
- WebRTC

### Authentication
- JWT + bcrypt

### Deployment
- Netlify (Frontend)
- Render (Backend)
